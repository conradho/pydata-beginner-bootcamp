Pydata Beginners' Bootcamp

Github Repo

Our Github repo includes this page, a README.md, all the notebooks, and more.

Notebooks

  • Data Visualization Notebook (html, ipynb): the least challenging notebook if you don't know where to start from
  • Geospatial Notebook (html, ipynb): Longitude/Latitude coordinates, nearest neighbors, and map visualizations
  • Quantitative Finance Notebook (html, ipynb): an investigation into optimal sizing
  • Natural Language Processing Notebook (html, ipynb): Twitter word count + clusters, sentiment analysis, and a Shakespeare generator

Html files include sample output; ipynb files are stripped of output.

For the ipynb files, right click and "save as"- if you click into the link and save the page, then you may get a html version of the .ipynb file, which jupyter won't recognize as a valid ipynb file.
